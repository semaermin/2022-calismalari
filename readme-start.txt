# Hastane-Otomasyonu
Kurulum adımları:
XAMPP’yi kurun.
Hastane klasörünü C:\xampp/htdocs klasörüne kopyalayın
Veritabanını Kurmak:

XAMPP uygulamasını çalıştırın (Apache ve mysql’nin çalıştığından emin olun).
Tarayıcıyı açın(google chrome, mozilla firefox).
“localhost/phpmyadmin” URL’sini yazın.
Hastane adıyla DB oluşturun.
Oluşturulan DB’ye tıklayın ve sql(hospital.sql) dosyasını içe aktarın
Sisteme erişim:
XAMPP uygulamasını çalıştırın (Apache ve mysql’nin çalıştığından emin olun).
Tarayıcıyı açın(google chrome, mozilla firefox).
“localhost/hastane” URL’sini yazın.
